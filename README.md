# Тестовое задание: PHP + Smarty + MySQL блог

Полностью рабочий блог на чистом PHP без фреймворков.

## Что реализовано

- Главная страница:
  - вывод категорий, в которых есть статьи;
  - по 3 последних статьи в каждой категории;
  - кнопка `Все статьи`.
- Страница категории:
  - название и описание категории;
  - список статей;
  - сортировка (`по дате`, `по просмотрам`);
  - пагинация.
- Страница статьи:
  - полная информация о статье;
  - 3 похожие статьи по совпадающим категориям.
- Сидинг:
  - создание таблиц;
  - заполнение тестовыми категориями и статьями.

## Стек

- PHP 8.1+
- Smarty
- MySQL
- Docker 
- SCSS 

Запуск встроенного PHP-сервера:

```bash
php -S localhost:8080 -t public
```

Откройте: [http://localhost:8080](http://localhost:8080)

---

## Запуск через Docker

1. Зависимости на хосте (один раз, без образа `composer`):

```bash
composer install
```

2. Поднять контейнеры:

```bash
docker compose up -d --build
```

3. В контейнере `.env` не нужен для БД — в `docker-compose.yml` уже заданы переменные. Сидинг:

```bash
docker compose exec app php database/seed.php
```

4. Сайт: [http://localhost:8080](http://localhost:8080)

MySQL снаружи контейнера: порт **3307** → `127.0.0.1:3307`, логин `blog` / пароль `blog`, БД `blog`.

## Структура

- `public/` — точка входа и статика
- `src/Core` — ядро (Router, Database, View, Env)
- `src/Controllers` — контроллеры страниц
- `src/Repositories` — запросы в MySQL
- `templates/` — Smarty шаблоны
- `database/` — SQL схема и сидер
- `assets/scss/` — SCSS исходник
- `public/assets/css/` — готовый CSS

## Команды

- сидинг:

```bash
php database/seed.php
```

или

```bash

```
